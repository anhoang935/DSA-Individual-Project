How to Use?

Welcome to my calendar and to-do-list application. Let me guide you through its features:

Calendar (Left Side)

  • Displays the month, year, and navigation arrows to move between months.
  • Select a specific day to mark it as active.
  • Days with events are indicated differently.
  • Clicking on a day with events shows the events in the to-do-list (right side)
  • Input month and year at here would move the calendar to that time.
  • Press Today button to jump current date on calendar to present date

To-Do-List (Right Side)

  • Displays the day of the week, date, month, and year for the active day.
  • Lists events for the selected day. If no events, it displays "No events".
  • Adding an event requires filling in event details (title, start time, end time).
  • Once submitted, events appear in the list. Clicking an event marks it as done.

Managing Event Lists

  • Clicking "Manage List" shows all other button feature.
  • Filtering (by month or year)
  • Sorting (ascending or descending)
  • Searching
  • Sorting or searching applies only after filtering events.
  • Clicking an event in manage form navigates the calendar to its date.
  • Note: You have to use filter button before use searching or sorting. Searching and sorting do not work together they will only return the event list base on the filter function
